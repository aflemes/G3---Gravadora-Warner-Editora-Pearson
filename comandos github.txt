1º Executar o git bash here na pasta que deseja commitar 

2º Adicionar as alterações ex:
  git add * (irá adicionar todos os arquivos da part)
  
3º Commitar as alterações ex:
  git commit -m "mensagem do commit"

4º Publicar o commit ex:
  git push origin master

5º Atualiazar as fontes na pasta exemplo:
  git pull origin master
